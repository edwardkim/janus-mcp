# Privacy Policy

**Effective Date:** November 29, 2025
**Last Updated:** November 29, 2025

## Overview

Janus MCP Converter ("the Software") is an open-source file conversion tool that operates entirely on your local machine. This privacy policy explains how the Software handles your data.

## Data Collection

**We do not collect any data.**

The Software:
- ✅ Runs 100% locally on your device
- ✅ Does not transmit any data to external servers
- ✅ Does not require an internet connection to function
- ✅ Does not collect usage statistics or analytics
- ✅ Does not store any personal information
- ✅ Does not use cookies or tracking technologies

## File Processing

When you use Janus MCP Converter:
- Files are processed entirely on your local machine
- Input files (PDF/TIFF) are read from your local filesystem
- Output files are written to your local filesystem
- No copies of your files are made except the intended output
- No file content is transmitted over the network

## Third-Party Services

The Software does not integrate with any third-party services that collect data.

## Network Access

The Software does not require or use network access for its core functionality. The only network access that may occur is:
- During initial installation via npm (downloading the package)
- This is handled by npm, not by the Software itself

## Open Source Transparency

Janus MCP Converter is open-source software licensed under MPL-2.0. You can review the complete source code at:
- npm: https://www.npmjs.com/package/@janus-mcp/converter
- GitHub: https://github.com/edwardkim/janus-mcp

## Children's Privacy

The Software does not collect any data from anyone, including children.
